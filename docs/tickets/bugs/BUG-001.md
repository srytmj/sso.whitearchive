# BUG-001 ‚Äî Guard `api` tidak terdefinisi di `config/auth.php`

**Severity**: P0 ‚Äî Blocker (GET /api/user selalu 500)
**Status**: Fixed ‚úÖ
**Found during**: Runtime verification session 2026-07-21

## Symptom

```
GET /api/user HTTP/500
{"message": "Auth guard [api] is not defined."}
```

## Root Cause

`config/auth.php` hanya mendefinisikan guard `web`. Route `GET /api/user` menggunakan middleware `auth:api`, tapi guard `api` tidak ada.

## Fix Applied

Tambahkan guard `api` di `config/auth.php`:

```php
'api' => [
    'driver' => 'passport',
    'provider' => 'users',
],
```

**File**: `config/auth.php` ‚Äî sudah difix di session ini.
